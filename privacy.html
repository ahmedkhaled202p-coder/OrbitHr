سياسة الخصوصية

آخر تحديث: 2026-07-14

هذه صفحة قالب قابلة للتعديل قبل النشر الرسمي. يجب مراجعتها قانونيًا وربطها بسياسة الخصوصية النهائية للشركة.

البيانات التي يعالجها التطبيق

يعالج Orbit HR بيانات الموظفين، الحضور والانصراف، الإجازات، الرواتب، المستندات، العهد، الطلبات، وسجل العمليات داخل النظام.

مكان التخزين

النسخة الحالية تحفظ البيانات محليًا داخل متصفح الجهاز المستخدم. عند تحويلها إلى نسخة إنتاجية متعددة المستخدمين يجب تحديد الخادم وقاعدة البيانات وسياسات الحماية والنسخ الاحتياطي.

الصلاحيات

قد يحتاج التطبيق إلى صلاحيات الكاميرا أو الموقع عند تفعيل التحقق بالحضور. لا يتم استخدام هذه الصلاحيات إلا لغرض تسجيل الحضور والتحقق وفق إعدادات الشركة.

مشاركة البيانات

لا تشارك النسخة المحلية البيانات مع أطراف خارجية تلقائيًا. أي تكاملات مستقبلية مع خادم أو أجهزة بصمة أو خدمات خارجية يجب توضيحها هنا.

التواصل

للاستفسار عن الخصوصية، أضف بيانات التواصل الرسمية الخاصة بالشركة قبل النشر.

الكاميرا والموقع: يستخدم التطبيق إذن الكاميرا واللوكيشن فقط عند تسجيل الحضور والانصراف، ويخزن لقطة تحقق مصغرة وإحداثيات التسجيل داخل بيانات النظام حسب إعدادات الشركة.

إضافة v4.0 Server Edition

تدعم النسخة حفظ البيانات والمرفقات وسجلات الدخول والحضور والنسخ الاحتياطية داخل قاعدة بيانات السيرفر. عند استخدام GPS أو الكاميرا يتم حفظ بيانات التحقق طبقًا لإعدادات الشركة والصلاحيات.
