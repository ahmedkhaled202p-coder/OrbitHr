# Checklist نشر Orbit HR v4.1

- [ ] تشغيل خلف HTTPS.
- [ ] تحديد دومين رسمي.
- [ ] ضبط متغير `ORBIT_HR_ALLOWED_ORIGINS`.
- [ ] تغيير كلمات المرور التجريبية.
- [ ] تحديد مواقع كل الفروع.
- [ ] تفعيل Backup يومي.
- [ ] اختبار استرجاع Backup.
- [ ] مراجعة صلاحيات كل مجموعة.

## الموبايل

- [ ] تجهيز Capacitor أو TWA.
- [ ] Android package name ثابت.
- [ ] iOS Bundle ID ثابت.
- [ ] أيقونات وسكرينشوتات المتاجر.
- [ ] Privacy Policy منشورة على رابط HTTPS.
- [ ] Terms منشورة على رابط HTTPS.
- [ ] اختبار الكاميرا واللوكيشن على جهاز فعلي.
- [ ] تصريح استخدام الكاميرا واللوكيشن في Android/iOS.

## قاعدة البيانات

- [ ] SQLite مناسبة للتجربة والاستخدام المحدود.
- [ ] PostgreSQL/MySQL أفضل عند التشغيل متعدد الفروع والمستخدمين بكثافة.
- [ ] لا ترفع ملف `server_secret.key` أو قاعدة البيانات لمكان عام.
