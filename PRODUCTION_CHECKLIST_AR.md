# Checklist قبل الاستخدام الإنتاجي أو رفع المتاجر

- تغيير كلمات المرور التجريبية.
- تشغيل السيرفر على HTTPS.
- تحديد مواقع الفروع ونطاق السماح.
- اختبار تسجيل الحضور GPS/Camera من موبايل فعلي.
- ضبط الصلاحيات والمجموعات.
- تشغيل فحص الجودة من الإعدادات.
- إنشاء Backup Server قبل وبعد كل تغيير كبير.
- تجربة دورة الرواتب كاملة: مسودة → مراجعة → اعتماد → صرف → إغلاق.
- مراجعة Privacy Policy و Terms حسب بلد التشغيل.
- تجهيز Android/iOS Wrapper واختبار الأذونات.
