استخدام التطبيق

يستخدم Orbit HR لإدارة الموارد البشرية والحضور والرواتب والتقارير داخل الشركة أو المؤسسة المالكة للنظام.

مسؤولية البيانات

مسؤول النظام مسؤول عن صحة بيانات الموظفين والحضور والرواتب والصلاحيات والنسخ الاحتياطية.

النسخ الاحتياطي

يجب إنشاء نسخ احتياطية دورية، خاصة قبل أي تعديل كبير أو ترقية إصدار.

حدود النسخة الحالية

النسخة الحالية تعمل كتطبيق ويب محلي/تقدمي. الاستخدام الإنتاجي متعدد الأجهزة يحتاج خادمًا وقاعدة بيانات وصلاحيات وأمانًا مركزيًا.

التعديلات المستقبلية

يمكن تطوير النظام وربطه بتطبيقات الجوال وأجهزة البصمة وقواعد البيانات حسب احتياجات المؤسسة.

تسجيل الحضور: يتحمل مسؤول النظام ضبط مواقع الفروع ونطاق السماح، ويجب إبلاغ الموظفين بسياسة استخدام الكاميرا واللوكيشن قبل التفعيل.

إضافة v4.0 Server Edition

تدعم النسخة حفظ البيانات والمرفقات وسجلات الدخول والحضور والنسخ الاحتياطية داخل قاعدة بيانات السيرفر. عند استخدام GPS أو الكاميرا يتم حفظ بيانات التحقق طبقًا لإعدادات الشركة والصلاحيات.
