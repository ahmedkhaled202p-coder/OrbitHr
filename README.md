## إضافات الإصدار 3.7.9
- تحويل مجموعات الرواتب الشهرية إلى أقسام قابلة للفتح والقفل.
- عرض كل موظفي الشهر في جدول واحد تحت عنوان الشهر بدل التقسيم الداخلي حسب الفرع.
- تغيير عنوان الشهر من "مسير شهر" إلى "مرتبات شهر".

## إضافات الإصدار 3.7.6

- إضافة زر **حذف رواتب المحدد** داخل شريط الإجراءات في قسم الرواتب.
- بعد تحديد موظف واحد أو عدة موظفين من جدول الرواتب يمكن حذف مسيراتهم دفعة واحدة.
- يسمح بالحذف للحالات: **مسودة، تمت المراجعة، معتمد**.
- حماية الرواتب **المصروفة أو المغلقة** من الحذف، مع توضيح عدد السجلات التي تم تجاهلها.
- لا يتغير مفتاح التخزين المحلي، لذلك تبقى بيانات النسخة السابقة محفوظة في المتصفح نفسه.

## إضافات الإصدار 3.7.2

- إضافة فلتر مستقل لاختيار الموظف داخل قسم الرواتب.
- عند تحديد أكثر من راتب يظهر زر **تصدير رواتب المحدد PDF** داخل شريط الإجراءات الجماعية، ولا يظهر هذا الزر في الأقسام الأخرى.
- ملف PDF المحدد يعرض فقط الموظفين المختارين، مجمّعين حسب الفرع مع ملخص الاستحقاقات والخصومات والصافي.
- إعادة ضبط هوية البرنامج بألوان برونزية وذهبية متناسقة مع شعار أوربت.
- تحسين عرض الشعار في القائمة الجانبية، شاشة الدخول، قسيمة الراتب، والتقارير المطبوعة.

نظام موارد بشرية عربي/إنجليزي يعمل من المتصفح مع الحفاظ على بيانات الإصدارات السابقة.

## إضافات الإصدار 3.7.1

### إجراءات جماعية ذكية حسب القسم

- إضافة زر واضح باسم **تصدير PDF** داخل أدوات كل قسم.
- إزالة زر **PDF** المختصر الذي كان يظهر في الشريط العلوي.
- عند تحديد سجلات يظهر شريط إجراءات مناسب للقسم الحالي بدل تصدير PDF للمحدد.
- أمثلة: اعتماد أو رفض الإجازات، تفعيل أو إيقاف الموظفين، تحديث حالة الحضور، حذف المستندات، وإجراءات الرواتب والعهد حسب الصلاحية.
- عند فتح نافذة الطباعة اختر **حفظ كملف PDF / Save as PDF**.

### الرجوع إلى الصفحة نفسها بعد Refresh

- يحفظ النظام آخر قسم تم فتحه في الجلسة.
- عند تحديث الصفحة يعود المستخدم إلى القسم نفسه بدل الرجوع إلى لوحة التحكم.
- عنوان القسم يُحفظ أيضًا في رابط الصفحة باستخدام Hash لتسهيل الاسترجاع.

### رواتب مستقلة لكل فرع

- يجب اختيار الفرع قبل تشغيل مسير الرواتب.
- يتم احتساب موظفي الفرع المحدد فقط، ولا تُدمج الفروع في مسير واحد.
- عرض الرواتب في مجموعات منفصلة لكل فرع.
- كل مجموعة تعرض عملة الفرع، عدد الموظفين، إجمالي الاستحقاقات، الخصومات وصافي الفرع.
- تستمر معادلة الراتب في مراعاة الورديات والحضور والإجازات والمأموريات والإضافي والخصومات والتأمينات والضرائب.

### فلتر تاريخ موحّد

- إضافة فلتر **من تاريخ / إلى تاريخ** داخل أدوات كل قسم.
- الفلتر يعمل على السجلات التي تحتوي تاريخًا أو فترة زمنية.
- يمكن دمج فلتر التاريخ مع البحث والفلاتر المتخصصة الموجودة في الصفحات.

### تعديل الحالات من أي قسم

- إضافة زر **تعديل حالة** في الأقسام التي تحتوي سجلات لها حالة.
- اختيار السجل ثم الانتقال مباشرة إلى أي حالة متاحة.
- يدعم الرجوع من حالة **معتمد** إلى **تمت المراجعة** أو **قيد المراجعة**.
- يشمل الموظفين، الحضور، الإجازات، المأموريات، المصروفات، الخصومات والمكافآت، العهد، الرواتب والخدمات الإدارية.
- يتم تسجيل التغيير في سجل النظام.

## التشغيل

### Windows

شغّل `run.bat` ثم افتح العنوان الذي يظهر في المتصفح.

### macOS / Linux

```bash
./run.sh
```

أو:

```bash
python3 start_server.py
```

### نسخة ملف واحد

افتح `Orbit_HR_Demo_v3.7.html` مباشرة في Chrome أو Edge.
